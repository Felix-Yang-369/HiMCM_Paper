## 📌 Overview
This research explores the accelerating trend of atmospheric $CO_2$ levels. By utilizing multi-dimensional modeling, the project analyzes historical data, quantifies the greenhouse effect's intensity, and provides high-confidence projections for the year **2100**.

## 🧠 Modeling Approach
- **Artificial Neural Network (ANN)**: Built using MATLAB’s `fitnet` toolbox with Levenberg-Marquardt backpropagation (`trainlm`).
- **Time Series Analysis**: Identified deterministic seasonal cycles and long-term trends.
- **Correlation Study**: Established a mathematical coupling between $CO_2$ concentrations and global Sea/Land Surface Temperatures.

## 📁 Repository Contents
- `HiMCM_Paper.pdf`: The final technical report submitted for the competition.
- `Code/`: MATLAB scripts for neural network training and validation.
- `Data/`: Historical datasets from NOAA and the Mauna Loa Observatory.
